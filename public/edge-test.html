Help

About this tool

This tool helps test your Supabase Edge Functions with different authentication header formats.

API Key

The API key is used for authentication with the Edge Function. It will be tried with both the x-api-key header and the Authorization: Bearer header format.

URLs

You can test two different endpoints:

  • Edge Function URL: Direct Supabase Edge Function URL (e.g., https://[project-ref].supabase.co/functions/v1/[function-name])
  • Super Action URL: Your custom API endpoint that might forward to the Supabase Edge Function

Troubleshooting

If you encounter errors:

  • Check if your API key is correct
  • Verify that the Edge Function is deployed
  • Check the CORS configuration on the server
  • Look at the browser console for more detailed error messages
